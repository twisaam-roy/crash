This a 2D game made with help of HTML5, CSS, Javascript in assistance with PIXI JS V.8.4.1(current latest).
You can play the game by clicking the given buttons on the screen to change bets start game, cash out and reset the game.
The game is ready to go that means steps to play the game are as follows:
  1. clone or copy the repository to your local device using git commands or unzipping the WinRar or WinZip file
  2. run command "npm install" or "npm i" inside the project directory where index.html,package json are present.
  3. run command "npm start" to start the game on your localhost with port: 8080

There are some changes which need to be made for PRODUCTION purposes such as code refractoring and building the game through VITE and PIXI JS
